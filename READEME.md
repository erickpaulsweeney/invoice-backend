# Invoicing System Backend

## Features
- Login as User
- Create clients
- Create items or services offered
- Create Invoice for a client with multiple items
- Update Invoice status
- See Invoices based on status
- Download pdf copy of Invoices

### Models Created
- User
- Client
- Item
- Invoice

### Packages used
- bcryptjs
- cors
- dotenv
- express
- jsonwebtoken
- mongoose 
- morgan
- pdfkit
